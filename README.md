# FinanceCore

> **short_description:** Sistema financeiro pessoal assistido por IA com Clean Architecture, DDD e PostgreSQL;
>
> **full_description:** Finance Core é um módulo de controle financeiro pessoal projetado com arquitetura limpa e DDD leve, totalmente offline-first e com assistência por inteligência artificial
> 
Diferenciais técnicos:
- Domínio 100% isolado de frameworks (Clean Architecture)
- Entrada por linguagem natural via IA (Strategy Pattern)
- Persistência local segura com PostgreSQL + criptografia AES
- Modelagem baseada em agregados e regras de negócio formais
- Testes unitários e de integração como parte do DoD;
>
